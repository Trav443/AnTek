1. Do you keep any personal information?

We only store what we need, we dont log any browser metadata, we dont sell data to advertisers, and we dont share the we use data with anyone.

2. What's the purpose of this?

To try and show people how to protest their government in an effective and modern way.

3. What is this website about?

It's a website about a lot of things but mainly libertarianism and technology.

4. How private is this website?

We are a site built by hackers so we try and secure things as much as possible, while anything is hackable we try and keep things secure, salting passwords, sanitizing inputs and using reliable and frequently updated software.
